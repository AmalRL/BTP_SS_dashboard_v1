# 📊 Analytics Dashboard (BTP & SS)

This is a Streamlit-based analytics dashboard used to track user engagement, onboarding, and intervention performance.

---

## 🚀 Features

- 📈 Onboarding Metrics
- 👥 Total Users & Active Users
- 📊 WAU (Weekly Active Users)
- 🔥 Power Users Identification
- 🎯 Age ≤ 36 Months Tracking
- 📦 Intervention Delivery Rate
- 📅 Weekly Filtering (Sunday → Saturday)
- 🔀 Multi-page Dashboard (BTP & SS)

---

## 🛠 Tech Stack

- Python
- Streamlit
- Amazon Redshift
- Pandas
- Plotly

---

## 📊 Key Metrics

- Total Users
- Onboarded Users & %
- Avg Weeks Active
- Age ≤ 36 Months Users
- WAU & WAU %
- Power Users & %
